# 🚗 Used Vehicle Price Predictor

A Machine Learning based web application that predicts the resale price of used vehicles using Gradient Boosting Regression.

---

## 📌 Problem Statement
Estimating the resale value of used vehicles manually can be difficult and inaccurate. This project uses Machine Learning techniques to predict vehicle prices based on important vehicle details.

---

## 🎯 Objective
- Predict used vehicle resale prices
- Build an AI-powered prediction system
- Create an interactive Streamlit web application
- Apply Machine Learning algorithms to real-world problems

---

## 🚀 Features
✅ Vehicle Price Prediction  
✅ Interactive Streamlit UI  
✅ Real-Time Price Estimation  
✅ User-Friendly Interface  
✅ Fast & Accurate Predictions
